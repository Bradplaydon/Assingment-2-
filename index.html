Box Model

There are 4 types of Box Model features, these are: margin, border, padding and also the content:

Content - The content of the box, where text and images appear.

Padding - Clears an area around the content. The padding is transparent.

Border - A border that goes around the padding and content.

Margin - Clears an area outside the border. The margin is transparent.

These four features allow html web pages to be styled in a more professional way margin, border,padding and content can be styled by using css external or internal, they can be styled by changing their width, height pixels, this allows things within these features to be changed accordingly. Inline css is styled on the same line as the code for example:

<h1 style="color:red;margin-top:15px;">Welcome.</h1>

h1 is a header inline css is helpful as it allows you to see clearly what's styled on that line. Internal css is the <style></style> tags allows individual pages to be styled as css documents work over all web pages it also allows to style the visited, hover, active, and link color of an anchor tag.

Implementation of CSS

Inline CSS

Inline css has some disadvantages, this is because it mixes content(code) with the html code . This creates a hard to manage site. The inline style has to be used carefully, to use inline styles you use the style attribute in the relevant tag. Inline CSS can be used for many purposes for example when it comes to creating a new project it can be used for testing as it is a lot easier to work with, its much easier to just scroll up in the html code rather than change the actual the css. It is also very quick to change to fix if you have a problem with your css. Using inline CSS helps users and service providers.

The main advantage of CSS is lower HTTP requests, which means the website loads faster. However, inline CSS has lots of limitations, which is where external css has its advantages, if you wanted to style <h1> or <p> tags you would have to individually add style inside of them, this can prove to be unorganized, and more to deal with, therefore external css is very useful as you can style everything from one document.

External CSS

There are countless advantages for using external CSS. For example, using external CSS allows full control and design of a page, it allows you to style individual classes, containers, columns and so on. IT also keeps your website and content separate this means that it is much more or. It is much easier to reuse your CSS code if you have it in a separate file. It doesn't require you to type the same CSS code on every web page which exists, you can just simply have many pages which refer to a single CSS file with the <link href="css/”Your css name here”" rel="stylesheet" type="text/css"/>.

External CSS also reduces the loading times of a website as browsers will cache the CSS therefore making loading times quicker, however if your css is big and has lots of styling elements in it, it will increase the download time of the css for that webpage, therefore making the speed of the webpage to be slow. Furthermore when someone is searching for a website through a browser, SEO(Search Engine Optimization) it is important to use external css because it helps create readable pages which will have lots of content, this can aid in making a site become more visible.

Internal CSS

Internal CSS Internal CSS has more advantages over inline CSS. With internal stylesheets it is still within the html code however it is within its own tags so it is more organized they’re within <style></style> tags within the <head></head> tags, internal css is usually specifically used for one in particular html page, unlike inline css where it is used for specific lines of code.
