HTML and CSS

Html access css files by using a correct file structure there are two ways html can use css internal and also external css internal uses the <style> tag and external uses the <link> tag css (cascading style sheet) allows many things to be changed and edited, css also allows you to select a certain part of your html document for example you can select the title, head, body, div tags, containers this allows you to individually style the html code within these containers. External uses <link rel=”stylesheet” type=”text/css” href=/main.css /> link rel defines the file a stylesheet.
So href would be ”href=css/main.css” if the css file is called something different then change href according to filename. The advantages of using external css are that it allows us to be able to style more than just one page at a time, inline css is within the html code, for example <h1 style=”black”>Lorum Ipsum</h1> this means you don’t have to create a whole new document, or use <style> tags in the head of the html file.
